# Personal Coding Project

A personal portfolio repository showcasing coding projects across multiple programming languages and web technologies.

## 📚 Repository Structure

This repository is organized by programming language and technology:

```
Personal-Coding-Project/
├── HTML/           # HTML projects and web pages
├── JavaScript/     # JavaScript projects and scripts
├── Python/         # Python projects and scripts
├── .vscode/        # VSCode workspace settings
└── sessions/       # Session tracking files
```

## 🚀 Technologies

This repository includes projects in the following technologies:

- **HTML** - Web page structure and markup
- **JavaScript** - Client-side scripting and web applications
- **Python** - General-purpose programming and scripts

## 📋 Project Overview

Each folder contains projects related to its respective language:

### HTML Projects
Located in the `HTML/` directory, featuring web pages and HTML markup examples.

### JavaScript Projects
Located in the `JavaScript/` directory, featuring JavaScript scripts and interactive web applications.

### Python Projects
Located in the `Python/` directory, featuring Python scripts and applications.

## 🛠️ Setup & Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Tun140113/Personal-Coding-Project.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Personal-Coding-Project
   ```

3. Explore the different language folders to find projects of interest.

### Running Projects

- **HTML Projects**: Open `.html` files directly in a web browser
- **JavaScript Projects**: Can be run in a browser or with Node.js depending on the project
- **Python Projects**: Run with Python 3:
  ```bash
  python3 filename.py
  ```
